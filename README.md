# RC

RC — каркас проекта для сервиса реального времени с разделением на подсистемы.

## Подпроекты
- [backend](./backend/) — API сервис.
- [realtime](./realtime/) — WebSocket/real-time шлюз.
- [rtc](./rtc/) — медиа слой (SFU/TURN и смежные компоненты).
- [client](./client/) — клиентское приложение.
- [infra](./infra/) — инфраструктура и деплой.
- [docs](./docs/) — документация.
