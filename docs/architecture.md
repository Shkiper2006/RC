# Архитектура

Краткое описание основных сервисов платформы:

- **API** — основной HTTP API сервис для бизнес-логики и управления данными.
- **WS** — WebSocket сервис для обмена событиями в реальном времени.
- **SFU** — селективный форвардер медиа-потоков для RTC/видеоконференций.
- **TURN** — сервер ретрансляции для обхода NAT/фаерволов.
- **DB** — основная база данных для хранения состояния и сущностей.
- **Redis** — кэш и брокер сообщений для быстрой синхронизации.
- **Nginx** — обратный прокси и входная точка для маршрутизации трафика.
